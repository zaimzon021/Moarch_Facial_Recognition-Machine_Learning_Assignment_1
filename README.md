# Facial Recognition & Gender Classification (Assignment 1)

Hey! I'm Zaim, and this is my first Machine Learning assignment. This project takes facial recognition a step further by integrating a gender classification model (`gender_model.h5`), all wrapped up in a clean web application so you can test it directly in your browser. 

*[Drop your screen recording video right here so people can see the web app in action!]*

### How to run it:
* **Download the code:** Run `git clone https://github.com/zaimzon021/Moarch_Facial_Recognition-Machine_Learning_Assignment_1.git` and open the folder.
* **Get the data ready:** Make sure to extract the images from the `Data.rar` file so the model has data to work with!
* **Install and Run:** Run `pip install -r requirements.txt` to grab the tools, then start the web app by typing `python app.py` in your terminal.
